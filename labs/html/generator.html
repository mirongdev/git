Generator Kode HTML Menu Navigasi

Keterangan
URL = Isi dengan URL tujuan (tujuan ketika menu diklik)
Nama Menu/Submenu = Isi dengan judul menu

Jika ingin menunya diklik mengarah ke halaman daftar postingan dengan label tertentu, isi URL dengan URL label.

Format URL label seperti ini :
https://ALAMATBLOG.COM/search/label/NAMALABEL

Contoh :
https://linkmagz.sugeng.id/search/label/Info
Cara Pemasangan Kode
  1. Salin kode menu di atas
  2. Masuk ke menu "Tata Letak" > edit widget "Kode Menu Navigasi"
  3. Letakkan kodenya di bagian "Konten" widget.
